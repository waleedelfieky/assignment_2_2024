This R\+O\+S-\/based application implements an action client that sends goals to a robot to navigate to a target location. The client communicates with the action server to control the robot\textquotesingle{}s movement and monitors its state through feedback. It also publishes the robot\textquotesingle{}s position and velocity as custom messages.\hypertarget{md__r_e_a_d_m_e_autotoc_md10}{}\doxysubsubsection{Key Features\+:}\label{md__r_e_a_d_m_e_autotoc_md10}

\begin{DoxyItemize}
\item {\bfseries{Action Client}}\+: Sends navigation goals (x, y) to the robot.
\item {\bfseries{Goal Control}}\+: Allows the user to set, cancel, or terminate navigation goals through command-\/line input.
\item {\bfseries{Robot State Publishing}}\+: The robot\textquotesingle{}s position and velocity are published in the {\ttfamily Robot\+State} message format, based on {\ttfamily /odom} data.
\item {\bfseries{Feedback and Status}}\+: Provides real-\/time feedback on the state of the goal (active, completed, or canceled).
