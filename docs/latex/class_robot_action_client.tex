\doxysubsection{Detailed Description}
A R\+OS action client that sends navigation goals to the robot and publishes its current state. 

This class connects to the \textquotesingle{}reaching\+\_\+goal\textquotesingle{} action server, allows sending and canceling goals, listens to the robot\textquotesingle{}s odometry, and publishes the robot\textquotesingle{}s current state (position and velocity) on the \textquotesingle{}robot\+\_\+state\textquotesingle{} topic.
