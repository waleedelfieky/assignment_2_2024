Subscribes to\+: ~\newline
 /reaching\+\_\+goal/goal — receives the goal target position for the robot. ~\newline


Services\+: ~\newline
 /get\+\_\+target — provides the latest target position (x, y) requested by the user. ~\newline


Description\+:

A service node that, when called, returns the coordinates of the last target sent by the user. It subscribes to the /reaching\+\_\+goal/goal topic to update the target coordinates and offers a service that allows clients to retrieve the latest target position.
