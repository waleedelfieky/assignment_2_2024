Detailed Description

Service node to provide target position of the robot.

Author
Elfieky Waleed
Version
0.1
Date
24/03/2025

Subscribes to:
/reaching_goal/goal — receives the goal target position for the robot.

Services:
/get_target — provides the latest target position (x, y) requested by the user.

Description:

A service node that, when called, returns the coordinates of the last target sent by the user. It subscribes to the /reaching_goal/goal topic to update the target coordinates and offers a service that allows clients to retrieve the latest target position.
